Основные команды терминала:
ls - cписок файлов и папок
флаг -а отобразит все файлы, включая скрытые
флаг -l отобразит подробные данные о каждом элементе

cd - текущая папка и перемещние между ними
cd .. - перенесет назад

pwd - отобразит полный путь до места где вы находитесь

mkdir - создать папку с названием которые вы напишете после команды. Наприме - mkdir test создаст папку с названием test

rm, rmdir - удалить файл/папку. Например - rmdir test удалит папку test

open - откроет папку/файл/программу и тп

cat - выводит в терминал содержимое .тхт файла

touch - создает файл с названием и расширением которое вы укажете. Например touch test.txt создаст текстовый документ с названием test

clear - очистить окно терминала

grep - найти текстовую строку в файле

vim - откроет редактирование выбранного файла через вим. Этой командой также можно заменить команду touch. Например если в пустой директории прописать vim test.txt, то у вас создастся указанный файл и сразу можно будет туда что-то записать

man - выводит мануал разработчика к указанной команде. Например git man выведет мануал к команде git
